# Pokedex

Creando un Pokedex con React y Typescript


## Tech Stack

- React
- TypeScript
- TailwindCSS
- Framer-Motion
- Chart.js

## Run Locally

Clonar el proyecto

```bash
  git clone https://github.com/Sakjara/pokedex2022.git
```

Ir al directorio del proyecto

```bash
  cd pokedex
```

Instalar dependencias

```bash
  npm install
```

Arrancar el server

```bash
  npm run start
```
